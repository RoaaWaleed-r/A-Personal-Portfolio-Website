Bluetooth controlled RC Car using Arduino

An educational project that combines electronics, programming, and remote control. It involves building a small-scale car equipped with a Bluetooth module and an Arduino microcontroller. The car can be controlled remotely using a smartphone or other Bluetooth-enabled device.

Traffic light controller using PIC16F877A

A Traffic light using 2 switches: one for Manual/Automatic and one for switching between 2 streets in Manual mode.
The timing in Automatic mode: West street (15s Red, 3s Yellow, 20s Green), South street (23s Red, 3s Yellow, 12s Green). By converting to manual mode, apply the 3s of Yellow.

Stereo Amplifier

A stereo audio amplifier designed using two LM386 integrated circuits with some features as Volume Control, Clean Sound, Flexible Power, Improved Sound Quality, Clearer Signal, Perfect Speaker Match, Power Indicator.

Tic Tac Toe using logic gates

A digital implementation of a Tic-Tac-Toe game. It uses logic gates to represent the game board, track player moves, and determine the winner or tie. The circuit likely includes inputs for player moves, a display to show the board state, and logic to check for winning conditions.

Voice Signal

A voice display project includes two waveform graphs, one showing the original audio signal and the other showing a filtered version of the signal. There are also several controls and indicators on the front panel, including buttons for acquiring sound and playing waveforms, as well as displays for displaying measurement data such as amplitude, frequency, and duration.

Network on Cisco packet tracer

A network topology diagram consisting of multiple devices, including PCs, switches, and routers with some features as Routing and Telnet.

Auto gate using Arduino

A gate that starts to close or open automatically when it senses the approach of any object.

Measuring Distance using Arduino

A project has been designed to measure the distance to an object using the ultrasonic sensor and display the measured distance on the LCD.
